## Лабораторна робота №1 — Семантична розмітка та мультимедіа

**Тема:** Семантична розмітка, мультимедіа та проектування інтерфейсів
**Мета:** Створити логічний архітектурний каркас веб-застосунку та інтерфейс збору даних за допомогою HTML5.

### Використані технології
- **HTML5 Semantic Markup**: `<header>`, `<nav>`, `<main>`, `<footer>`, `<article>`, `<section>`, `<figure>`.
- **Multimedia Optimization**: `<picture>` (webp/jpg), `<video>` з атрибутами розміру, `loading="lazy"`.
- **Performance**: Оптимізація LCP через `preload` та `fetchpriority`, стабілізація CLS через атрибути `width`/`height`.

## Лабораторна робота №2 — CSS стилізація

**Тема:** Типографіка, кольорова палітра, Box Model, інтерактивні елементи та навігація
**Мета:** Створити єдину систему стилів для трьох сторінок проєкту через зовнішній файл `css/style.css`.

### Завдання 1 — Типографіка та колір
- CSS-змінні для палітри: `--ink` (#1a0f0a), `--copper` (#c4956a), `--parchment` (#e8d5b7), `--cream` (#f2e8d5), `--copper-dim` (#8a6347), `--ink-soft` (#2d1f17).
- Шрифти через Google Fonts: **Playfair Display** (заголовки h1-h3) та **Lora** (body, форми).
- Одиниці виміру: `rem` для розмірів шрифту, `em` для відступів.
- Ієрархія заголовків: h1 — 2.8rem/900, h2 — 2rem/700, h3 — 1.4rem/600 з різним `letter-spacing`.

### Завдання 2 — Box Model та Layout
- Глобальний `box-sizing: border-box`.
- Клас `.container` з `max-width: 1200px`, центруванням та боковими відступами.
- Картки фільмів з `border-radius: 4px`, `overflow: hidden` та теплими тінями `box-shadow`.

### Завдання 3 — Форма та Таблиця
- Поля введення: тільки нижня рамка `border-bottom`, прозорий фон, плейсхолдери курсивом через `::placeholder`.
- Плавні переходи `transition` для фокусу на полях.
- Кнопка submit зі зміною кольору та `transform: translateY(-2px)` при наведенні.
- Таблиця розкладу: `border-collapse`, ефект зебри через `:nth-child(even)`, заголовки `<th>` з `uppercase` та розрядкою літер.

### Завдання 4 — Навігація
- Список меню `ul` з обнуленими стилями, `li` через `display: inline-block`.
- Усунення пробілів між елементами через `font-size: 0` на батьківському елементі.
- Посилання `<a>` з нижньою лінією при hover та активному стані `.active`.

### Структура файлів
```
css/
  style.css       /* головна таблиця стилів для всіх сторінок */
index.html        /* головна сторінка з прем'єрою та скоро в кіно */
catalog.html      /* каталог фільмів за жанрами та розклад сеансів */
contacts.html     /* форма зворотного зв'язку */
```
