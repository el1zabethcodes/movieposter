# КіноАфіша

**Виконавець:** Павлова Єлизавета, П-31

---

## Про проєкт

Веб-застосунок афіші кінотеатру з трьома сторінками: головна, каталог фільмів та контакти. Реалізовано адаптивний дизайн з підтримкою мобільних, планшетів та десктопів. Використовується семантична HTML5-розмітка, Tailwind CSS для стилізації та JavaScript для обробки даних про фільми.

---

## Технології

- **HTML5** — семантична розмітка, відео, lazy loading, оптимізація LCP.
- **CSS3** — кастомні змінні, Flexbox, CSS Grid, медіа-запити, темна тема (prefers-color-scheme).
- **Tailwind CSS** — утилітарні класи, адаптивні брейкпоінти md та lg.
- **JavaScript** — робота з масивами об'єктів, колекції Set та Map, доступ до полів через квадратні дужки.

---

## Структура файлів

```
css/
  style.css       /* Базові стилі */
  style3.css      /* Адаптивні стилі просунутого рівня */
js/
  script.js       /* JavaScript: обробка фільмів, масиви, Set, Map */
index.html        /* Головна сторінка з прем'єрою та анонсами */
catalog.html      /* Каталог фільмів за жанрами та розклад сеансів */
contacts.html     /* Форма зворотного зв'язку */
```

---

## Як запустити

Для перегляду відкрийте будь-який HTML-файл у браузері або запустіть локальний сервер:

```bash
python -m http.server 8080
```

Сайт буде доступний на http://localhost:8080

## Гіт-теги

- `ЛР_2_Просунутий_рівень` — CSS-стилізація: типографіка, Box Model, форми, таблиці, навігація.
- `пр1_просунутий_рівень` — адаптивний дизайн, mobile-first, Flexbox, Grid, бургер-меню.
- `пр2_просунутий_рівень` — інтеграція Tailwind CSS, утилітарні класи.
- `лр3_просунутий_рівень` — JavaScript: масиви, об'єкти, Set, Map.
