# КіноАфіша — Лабораторна робота №1

**Тема:** Семантична розмітка, мультимедіа та проектування інтерфейсів
**Мета:** Створити логічний архітектурний каркас веб-застосунку та інтерфейс збору даних за допомогою HTML5.

## Опис проекту
Веб-застосунок «КіноАфіша» розроблений для перегляду актуального розкладу фільмів, анонсів прем'єр та зворотного зв'язку з користувачами.

### Основні сторінки:
- `index.html` — Головна сторінка з головною прем'єрою та анонсами.
- `catalog.html` — Каталог фільмів з детальною таблицею розкладу сеансів.
- `contacts.html` — Форма зворотного зв'язку з валідацією даних.

### Технології:
- HTML5 (семантична розмітка)
- Мультимедіа (`<video>`, `<picture>`)
- Форми та валідація
