# КіноАфіша

**Виконавець:** Павлова Єлизавета, П-31

---

## Про проєкт

веб застосунок афіші кінотеатру з трьома сторінками головна каталог фільмів та контакти реалізовано адаптивний дизайн з підтримкою мобільних планшетів та десктопів використовується семантична html5 розмітка tailwind css для стилізації та джаваскрипт для обробки даних про фільми

---

## Технології

- **html5** семантична розмітка відео lazy loading оптимізація lcp
- **css3** кастомні змінні флексбокс css грід медіа запити темна тема prefers color scheme
- **tailwind css** утилітарні класи адаптивні брейкпоінти мд та лджі
- **javascript** робота з масивами обєктів колекції сет та мап доступ до полів через квадратні дужки

---

## Структура файлів

```
css/
  style.css       /* базові стилі */
  style3.css      /* адаптивні стилі просунутого рівня */
js/
  script.js       /* джаваскрипт обробка фільмів масиви сет мап */
index.html        /* головна сторінка з прем'єрою та анонсами */
catalog.html      /* каталог фільмів за жанрами та розклад сеансів */
contacts.html     /* форма зворотного зв'язку */
```

---

## Як запустити

для перегляду відкрийте будь який html файл у браузері або запустіть локальний сервер

```bash
python -m http.server 8080
```

сайт буде доступний на http://localhost:8080

## Гіт теги

- `пр1_просунутий_рівень` — адаптивний дизайн мобайл ферст флексбокс грід бургер меню
- `пр2_просунутий_рівень` — інтеграція tailwind css утилітарні класи
- `лр3_просунутий_рівень` — джаваскрипт масиви обєкти сет мап
